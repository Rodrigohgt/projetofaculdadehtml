Quem somos?

Olá!

A LandFull Run é uma associação que atua desde 2020 para trazer alegria e conforto para pessoas da terceira idade,
contamos com diversos cursos e viagens para melhor conforto de nossos associados, Na LandFull Run tudo foi cuidadosamente pensado para proporcionar maior conforto, segurança e qualidade de vida que nossos idosos merece.

O pequeno grupo fundador do LandFull Run criamos uma verdadeira teia solidária. Com isso, o LandFull Run se tornou um espaço bem estruturado para dar aconchego para seus associados. É uma referência de trabalho social.
A diretoria da LandFull Run é composta por voluntários que trabalham e buscam melhorias para servir a todos.

A Associação LandFull Run é uma associação sem fins lucrativos,
fornece cursos e viagens para todo o Brasil.
É um associação voltada para a terceira idade.
